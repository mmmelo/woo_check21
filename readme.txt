This plugin ability for each merchant to communicate and transfer checking transactional information securely to the Check 21 system

== Installation ==
* WordPress 3.8 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Manual installation =
The manual installation method involves downloading our eCommerce plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

* Download plugin file
* Copy folder to wp-content/plugins
* Active the plugin
* Set your Check21 Credentials Info on Woocommerce -> Settings -> Checkout -> HighTechWeb Check21

= Required: =
* Need Account on [Check21.com](http://check21.com/)
* Installed [WooCommerce](http://wordpress.org/extend/plugins/woocommerce/)
